# Enterprise Network Design

## Objective
To design a small-to-medium enterprise network with proper IP addressing,
VLAN segmentation, and inter-VLAN communication.

## Tools Used
- Cisco Packet Tracer

## Key Configurations
- Designed logical and physical network topology
- Implemented VLSM-based IP addressing
- Configured VLANs and inter-VLAN routing
- Planned routing between network segments

## What I Learned
- Importance of proper IP address planning
- Practical use of VLANs in enterprise environments
- How routing enables communication between network segments
