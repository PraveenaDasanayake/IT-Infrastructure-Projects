# Autonomous Maze-Solving Robot

## Objective
To develop an autonomous robot capable of navigating a maze using sensor input
and predefined logic.

## Tools & Technologies
- Arduino
- Ultrasonic Sensors
- Embedded C

## Key Features
- Implemented Left-Hand Rule algorithm for navigation
- Real-time obstacle detection using ultrasonic sensors
- Autonomous decision-making without manual control

## What I Learned
- Applying algorithms to real-world hardware
- Sensor integration and control logic
- System-level thinking and debugging
